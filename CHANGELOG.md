# Changelog

## 0.1.0 - Public Demo Release

- Added the first public-facing documentation.
- Documented quick-start commands and supported characters.
- Added explicit safety and ethics disclaimer.
- Added MIT license.
- Added package metadata and a `grid-graffiti` console script entry point.
- Added zero-dependency `unittest` coverage for rendering, date mapping,
  exports, safety checks, and local Git repository generation.
- Made the CLI entry point easier to test and hardened export parent-directory
  creation.
- Added `--author-name` and `--author-email` options for generated commit
  contributor attribution, defaulting to current Git config when available.
- Included sample `HELLO` artifacts:
  - `grid-preview.svg`
  - `grid-dates.csv`
  - `pixels.txt`
